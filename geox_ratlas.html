NOTES
ρb — Bulk Density (Gardner)
Linear mixing law:
ρb = (1 − φ) · ρm + φ · ρf
ρb bulk density [g/cc] φ porosity [v/v] ρm matrix density ρf fluid mix density
Water: ρf ≈ 1.0 · Oil: ≈ 0.85 · Gas: ≈ 0.001–0.3
Typical fluid mixes: SW × ρw + (1−SW) × ρhc
NPHI — Neutron Porosity (Hydrogen Index)
Fluid component mixing:
NPHI ≈ φ · Σ(Si · HIi)
HIw = 1.0 (bound water) HIhc ≈ 0.95–1.0 (oil) HIgas ≈ 0.25–0.35 (low-pressure gas) Si = saturation fraction
Matrix effect: limestone scale · Subtract density porosity from NPHI for gas effect
RT — Resistivity (Archie, 1942)
Clean sand / carbonate:
Rt = a · Rw / (φm · Swn)
a = 0.62–1.0 (cementation) m = 1.5–2.2 (cementation exponent) n = 1.8–2.5 (saturation exponent) Rw = formation water resistivity
VSH — Shale Volume (Gamma Ray Linear)
GR index method:
Vsh = (GRlog − GRmin) / (GRmax − GRmin)
GRlog = measured API GRmin = clean sand baseline GRmax = shale baseline
Clavier (1971) correction: Vsh = 1.7 − √(0.6 + ...)
Larionov (1969) older rocks: Vsh = 0.33(2IGR − 1)
